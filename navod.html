Návod - Tabulka s dynamickými sloupci a řádky

Autor: Pavel Macenauer (xmacen02)

Instalace

  1. Nakopírovat dynamicTable.js do složky s Javascript soubory a default.css do složky s CSS soubory.
  2. Nalinkovat soubor ve Vámi používaném souboru těsně před koncem tagu body, např. takto:
    ...
    <link rel="stylesheet" type="text/css" media="screen" href="css/default.css">
    </head>
    ...
    <script type="text/javascript" src="js/dynamicTable.js"></script>
    </body>
    </html>
  3. Vytvořit tabulku s parametry id a class, kde id je jedinečný identifikátor tabulky a pomocí class je dán její vzhled. V balíku je přiložen vzhled default, který lze použít jako obsah proměnné class. Následně ji inicializovat pomocí metody dynamicTable.setTable(id) dle jejího ID po načtení skriptu, např. takto:
    <table id="myTable" class="default">
    ...
    </table>
    ...
    <script type="text/javascript" src="js/dynamicTable.js"></script>
    <script type="text/javascript">
    dynamicTable.setTable("myTable");
    </script>
    </body>
    </html>

Výchozí vzhled

  • Vzhled tabulky je nadefinován soubory ve složce css, konkrétně v souboru default.css, který je generovaný ze souborů default.scss a _variables.scss pomocí preprocessoru SASS. Pokuď SASS nepoužíváte lze editovat přímo default.css, nicméně je silně doporučeno měnit pouze obsah proměnných v souboru _variables.scss a překompilovat následně vygenerovaný default.scss, popř. pokuď máte dostatek odvahy, tak i samotný default.scss.

Co to umí?

  • Tažením za zvýrazněnou část v hlavičkových buňkách lze měnit šířku a výšku jednotlivých sloupců a řádku, kromě hlavičky.
  • Přetažením hlavičkových buněk lze prohazovat sloupce.
